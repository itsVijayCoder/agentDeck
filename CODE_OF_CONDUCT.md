# Code of Conduct

AgentDeck uses a simple contributor standard: be respectful, direct, and constructive.

## Expected Behavior

- Discuss ideas and code on their technical merits.
- Assume good intent while still asking for evidence and clarity.
- Give actionable feedback.
- Respect different experience levels, backgrounds, and working styles.
- Keep disagreements focused on the work.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated disruption of technical discussion.
- Any behavior that makes the project unsafe or unwelcoming for contributors.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior conflicts with this code of conduct.

If you need to report a concern, contact the maintainers through the repository's preferred private channel once it is published.
